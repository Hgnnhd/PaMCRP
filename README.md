This repository contains the code for PaMCRP, a deep learning framework designed for passive, multi-cancer risk prediction using longitudinal Electronic Health Records (EHR). PaMCRP leverages disease trajectories (sequences of ICD-10 codes, age at diagnosis, time intervals) and employs a transformer-based architecture with hierarchical embedding and prototype learning.

A key feature is its synergistic learning approach, where the model is trained simultaneously on data from the five highest-incidence cancers (Lung, Breast, Colorectal, Prostate, Gastric/Stomach). This allows the model to capture both cancer-specific and shared risk patterns, leading to improved predictive performance compared to single-cancer models. The framework operates passively, utilizing existing EHR data without requiring dedicated active screening tests.
